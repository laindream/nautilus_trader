# 投资组合示例

演示如何在 NautilusTrader 中使用投资组合的简单策略。

投资组合是跟踪交易账户状态的核心组件。
它直接连接到经纪商以获取实时持仓、余额和损益。

## 示例亮点

该策略在四个关键点显示投资组合信息：

1. **初始状态**：在执行任何交易之前。
2. **持仓开启**：当创建新持仓时。
3. **交易中期**：持仓开启两分钟后。
4. **最终状态**：所有持仓关闭后（策略停止时）。

为了模拟这些特定的投资组合状态，策略触发括号订单（入场订单与相关的止盈和止损订单的组合），使我们能够演示投资组合状态的完整生命周期。

## 附加信息

`投资组合` 和 `缓存` 之间的主要区别：

`投资组合`：

- 直接从经纪商获取数据以确保最大准确性。
- 最适合实时持仓和风险管理。
- 提供权威的账户状态（保证金、余额）。
- 应用于关键交易决策。

`缓存`：

- 将所有交易数据存储在系统内存中。
- 适用于快速访问历史数据和市场状态。
- 频繁查询时更高效，因为避免了经纪商往返。
- 新数据到达时自动更新。
- 与经纪商数据相比可能有最小延迟。
